# 🕵️‍♂️ Graph-RAG 기반 다자간 논리 추론 마피아 엔진 (v1.4)

이 프로젝트는 단순한 게임 시뮬레이션을 넘어, AI 에이전트들이 **그래프 데이터베이스(Neo4j)** 의 하드 팩트와 **벡터 데이터베이스(Milvus)** 의 대화 맥락을 결합하여 고차원적인 심리전과 논리 추론을 수행하는 **지능형 협업 에이전트 시스템**입니다.



---

## 1. 프로젝트 핵심 철학
본 엔진은 AI의 고질적인 문제인 **'망상(Hallucination)'** 을 방지하기 위해 두 종류의 외부 기억 저장소를 사용합니다.

1.  **그래프 DB (Neo4j) - "논리적 팩트"**: "누가 누구와 대립하는가?", "누가 경찰인가?" 등 변하지 않는 **관계 데이터**를 저장합니다.
2.  **벡터 DB (Milvus) - "대화의 맥락"**: "과거에 이 사람이 어떤 식으로 말했는가?"를 저장하여 **발언의 일관성**을 추적합니다.
3.  **하이브리드 RAG**: 위 두 데이터를 실시간으로 결합하여, 에이전트가 입을 열기 직전 **"현재 가장 모순된 상황"** 을 브리핑해 줍니다.

프로그램이 어떻게 돌아가는지 궁금하다면 아래 순서대로 파일을 읽어보세요.

| 모듈 명칭 | 파일 경로 | 주요 역할 (비유) |
| :--- | :--- | :--- |
| **[A] 그래프 매니저** | `database/neo4j_setup.py` | 게임 속 인물들의 인맥과 대립 관계를 기록하는 **'관계 장부'** |
| **[B] 벡터 매니저** | `database/milvus_setup.py` | 수천 마디의 대화 중 중요한 맥락을 기억해내는 **'기억 저장소'** |

## 3. 핵심 게임 시스템 (Core Logic)

### 🧠 2-Phase 전략 토론 (Strategic Debate)
단순히 한 번 말하고 끝나는 것이 아니라, 고도의 심리전이 가능하도록 설계되었습니다.
- **1단계(탐색전)**: 자신의 정체를 밝히고 초기 의심 대상을 던집니다.
- **2단계(결전)**: 상대방의 반박을 듣고 RAG 팩트를 대조합니다. 마피아는 불리하면 동료를 버리는 **'꼬리 자르기'**를, 시민은 정보에 따라 **'추리 수정'**을 수행합니다.

### ⚖️ 언행일치 투표 시스템
에이전트가 토론 중에 가장 마지막으로 마피아라고 지목(`ACCUSES_MAFIA`)한 대상에게 **실제로 투표**가 던져집니다. 논리와 행동의 무결성을 보장합니다.

### 🔍 정보의 비대칭성 (Information Advantage)
- **진짜 경찰**: 시스템으로부터 직접 조사 결과를 하사받아 **100% 정확한 정보**를 가집니다.
- **사칭 마피아**: 정보가 없으므로 무작위로 타겟을 찍어 '시민'이라고 거짓말합니다. 만약 찍은 사람이 실제 '의사'라면? **논리적 파멸**이 시작됩니다.

## 4. 개발 준수 프로토콜
모든 코드 수정은 아래 4가지 원칙을 엄격히 따릅니다.
- **무생략 원칙**: 수정 시 복사-붙여넣기가 가능하도록 전체 코드를 제공합니다.
- **연쇄 수정 명시**: 한 모듈의 수정이 전체 시스템에 미치는 영향을 주석으로 설명합니다.
- **논리 무결성**: 사망자 발생 시 역추론 로직을 통해 누가 가짜인지 시스템적으로 확정합니다.
- **교육적 주석**: 코드의 흐름과 구현 이유를 상세히 기술합니다.

## 5. 시작하기 (Quick Start)

### 1) 환경 설정
`.env` 파일에 OpenAI API 키와 DB 접속 정보를 입력합니다.
```env
OPENAI_API_KEY=your_key
NEO4J_URI=bolt://localhost:7687
NEO4J_USER=neo4j
NEO4J_PWD=password
MILVUS_ENDPOINT=https://your-milvus-endpoint
MILVUS_TOKEN=your-token
```

### 2) 패키지 설치
```
pip install -r requirement
```
### 3) 엔진 가동
```
python project/game_coordinator.py
```
