### 시민 승리 시나리오 분석 (Episode: Citizen Victory)

**1. 시스템 초기화 및 밤의 행동**
* Milvus 및 Neo4j 데이터베이스 초기화 및 6인 게임 세팅이 완료되었습니다.
* **밤의 살인**: Agent 2(마피아)가 Agent 3(시민)을 살해하였습니다.
* **경찰 조사**: Agent 6(경찰)이 Agent 3을 조사하여 시민임을 확인하였습니다.

**2. Day 1: 첫 번째 발언 (탐색전 및 대립 형성)**
* **Agent 1 (마피아/위장)**: 현재 정보가 부족함을 언급하며 Agent 2를 무작위 투표 대상으로 지목했습니다.
* **Agent 2 (마피아/경찰 사칭)**: 자신을 경찰로 주장하며 Agent 5가 시민임을 확인했다고 발표했습니다. 근거 없이 자신을 지목한 Agent 1을 역으로 의심했습니다.
* **Agent 4 & 5 (시민)**: Agent 2의 경찰 주장과 Agent 1의 무작위 지목 사이에서 혼란을 겪었으나, 일단 Agent 2의 논리에 동조하여 Agent 1을 주시했습니다.
* **Agent 6 (진짜 경찰)**: 실제 경찰임을 밝히며 간밤에 사망한 Agent 3이 시민임을 확인했다고 발표했습니다. 자신의 정체를 근거로 Agent 2의 주장이 거짓임을 폭로했습니다.

**3. Day 1: 두 번째 발언 (논리적 수렴 및 확정)**
* **Agent 1 (마피아/전략 수정)**: Agent 2와 Agent 6의 대립을 분석했습니다. 사망자(Agent 3)의 정체를 정확히 맞춘 Agent 6의 정보가 더 신뢰성이 높다고 판단하여 Agent 2를 타겟으로 변경했습니다.
* **Agent 2 (마피아/고립)**: Agent 6의 주장이 자신을 함정에 빠뜨리려는 시도라고 반박하며 Agent 6 처형을 주장했습니다.
* **Agent 4, 5, 6 (시민)**: RAG 시스템 팩트를 통해 Agent 6의 조사 결과가 실제 사망자 정보와 일치함을 확인했습니다. 반면 증거가 부족한 Agent 2를 마피아로 확정 지었습니다.

**4. 1차 투표 결과 및 정체 공개**
* **투표**: Agent 1, 4, 5, 6이 최종 결론에 따라 Agent 2에게 투표하였습니다.
* **처형**: 최다 득표를 얻은 **Agent 2**가 처형되었습니다.
* **정체 확인**: 사망한 **Agent 2의 실제 직업은 [마피아]**로 밝혀졌습니다.

**5. Day 2: 밤의 행동 및 최종 승리**
* **밤의 살인**: 남은 마피아인 Agent 4가 Agent 5(의사)를 살해하였습니다.
* **논리 확정**: 의사(Agent 5)가 사망하면서 기존 대립 관계에 있던 에이전트들에 대한 추가적인 역추론이 발생했습니다.
* **최종 토론**: Agent 6(경찰)은 Agent 1이 시민임을 조사로 확인했습니다. Agent 1과 Agent 6은 협력하여 남은 생존자 중 모순된 행동을 보인 Agent 4를 마피아로 지목했습니다.
* **결론**: 투표 결과 **Agent 4(마피아)**가 처형되었으며, 모든 마피아 소탕으로 **시민 팀이 최종 승리**하였습니다.
